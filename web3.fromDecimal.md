# web3.fromDecimal

将一个数字，或者字符串形式的数字转为一个十六进制串。

调用：
```
web3.fromDecimal(number)
```

参数：

- `number`：Number|String - 数字

返回值：

- `String` - 给定数字对应的十六进制表示。

示例：
```
var value = web3.fromDecimal('21');
console.log(value); // "0x15"
```
