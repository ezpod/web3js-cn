# web3.setProvider

设置网络通信服务提供对象。

调用：
```
web3.setProvider(provider)
```
参数：

- `provider`：通信服务提供对象

返回值：

- 无


示例：
```
web3.setProvider(new web3.providers.HttpProvider('http://localhost:8545'));
```
