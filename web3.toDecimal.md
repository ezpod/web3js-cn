# web3.toDecimal

将一个十六进制转为一个十进制的数字

调用：
```
web3.toDecimal(hexString)
```

参数：
- `hexString`：String - 十六进制字符串

返回：

- `Number` - 传入字符串所代表的十六进制值。

示例：
```
var number = web3.toDecimal('0x15');
console.log(number); // 21
```
