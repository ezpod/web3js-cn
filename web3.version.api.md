# web3.version.api

调用：
```
web3.version.api
```

返回值：

- `String` - 以太坊js的api版本

示例：

```
//test.js 省略初始化过程
var version = web3.version.api;
console.log(version);
```

在终端执行：
```
$ node test.js
0.18.2
```
